# Agent 逻辑流程与优化方案

## 当前逻辑流程

### 1. 用户输入
```
用户提问 → Agent 接收
```

### 2. Agent 决策
```
Agent 分析问题 → 决定使用哪个工具
```

### 3. 工具调用
```
Agent 调用工具 → 获取结果
```

### 4. 生成回答
```
Agent 基于工具结果 → 生成回答
```

---

## 问题分析

### 问题 1：知识库优先策略未执行

**现象**：
- 用户提问："如何通过jdbc方式连接gbase 8a数据库"
- 实际调用：web_search → fetch_url（3次）
- 缺失调用：❌ search_knowledge_base

**原因**：

1. **System Prompt 约束力不足**：
   ```yaml
   # 当前 System Prompt
   ## 回答策略（CRITICAL）
   ### 优先级顺序（严格遵循）
   1. 知识库检索（最高优先级）
   2. 联网搜索
   3. LLM 自身能力
   ```
   - 虽然强调了"最高优先级"，但没有强制要求
   - Agent 可以根据"最佳判断"跳过知识库检索

2. **Agent 决策逻辑不明确**：
   - Agent 没有明确的"必须先检索知识库"的规则
   - 可能认为联网搜索能更快得到结果

3. **工具调用顺序不固定**：
   - 没有强制要求工具调用的顺序
   - Agent 可以自由选择工具

### 问题 2：回答质量不稳定

**现象**：
- 回答长度不一致
- 回答深度参差不齐
- 信息来源不明确

**原因**：
- System Prompt 中的长度限制不够强制
- LLM 对"深度要求"的理解不一致
- 没有统一的信息来源标注规范

## 优化方案：混合检索

### 方案设计

```
用户输入
    ↓
[1] 知识库检索（GBase8a 官方文档）
    ├─ 搜索结果：top_k=5, min_score=0.5
    └─ 评分：基于相似度、来源权威性
    ↓
[2] 联网搜索（补充最新信息）
    ├─ 搜索结果：top_k=5
    └─ 评分：基于来源权威性、时效性
    ↓
[3] 结果融合（加权排序）
    ├─ 知识库结果：权重 70%（官方权威）
    ├─ 联网搜索结果：权重 30%（补充信息）
    └─ 去重、排序、截取 top_k=10
    ↓
[4] 生成回答
    ├─ 优先使用知识库信息
    ├─ 用联网搜索信息补充
    └─ 标注信息来源
```

### 技术实现

#### 1. 混合检索工具

```python
@tool
def hybrid_search(query: str, top_k: int = 10, dataset: str = "gbase8a", runtime: ToolRuntime = None) -> str:
    """
    混合检索：结合知识库检索和联网搜索，提供更全面的信息
    """
    # 1. 知识库检索（权重 70%）
    knowledge_results = _search_knowledge_base(query, dataset, top_k // 2)

    # 2. 联网搜索（权重 30%）
    web_results = _search_web(query, top_k // 2)

    # 3. 结果融合（加权排序）
    fused_results = _fuse_results(knowledge_results, web_results)

    # 4. 格式化输出
    return _format_results(query, fused_results)
```

#### 2. System Prompt 优化

```yaml
## 回答策略（CRITICAL - 强制执行）

### 工具调用顺序（必须遵循）

1. **第一步：混合检索**
   - 所有技术问题必须先使用 `hybrid_search` 工具
   - 参数：query（用户问题）, dataset="gbase8a"
   - 如果混合检索失败，标记为"检索失败"并进入下一步

2. **第二步：网页获取（如需要）**
   - 如果混合检索结果不足或需要更多细节
   - 使用 `fetch_url` 获取指定网页内容
   - 参数：url（网页地址）, detail_level="full"

3. **第三步：生成回答**
   - 基于检索结果生成回答
   - 优先使用知识库信息
   - 用联网搜索信息补充
   - 标注信息来源

### 回答规范（强制执行）

1. **长度控制**：
   - 简单问题：50-150 字
   - 中等问题：150-300 字
   - 复杂问题：300-500 字
   - 文档生成除外

2. **结构要求**：
   ```
   ## [核心答案]

   [详细说明/示例代码]

   [注意事项/最佳实践]

   **信息来源：[来源]**
   ```

3. **来源标注**：
   - 知识库：`信息来源：GBase8a 官方知识库`
   - 联网搜索：`信息来源：联网搜索`
   - 混合：`信息来源：GBase8a 官方知识库 + 联网搜索`
```
